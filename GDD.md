## 1. 프로젝트 개요 (Executive Summary)

### 1.1 게임 타이틀 및 장르

| 항목 | 내용 |
|------|------|
| **타이틀** | Tidehaven |
| **부제** | 낚시와 채굴의 조용한 마을 |
| **장르** | 멀티플레이어 낚시 RPG / 힐링 샌드박스 |
| **플랫폼** | 웹 브라우저 (PC / 모바일) |
| **엔진** | React + HTML5 Canvas + Firebase Realtime |

### 1.2 핵심 컨셉 한 줄 요약

> **"혼자서도, 함께해도 즐거운 — 낚시터 마을에서 자신만의 리듬으로 성장하는 힐링 멀티플레이 RPG"**

### 1.3 타겟 플레이어

| 세그먼트 | 설명 |
|---------|------|
| **핵심 타겟** | 18~35세, 가볍게 즐기는 힐링 게임 선호자 |
| **2차 타겟** | 방치형/낚시 시뮬레이션 팬, 소셜 MMO 입문자 |
| **플레이 패턴** | 짧은 세션(15~30분) 반복 or 장시간 방치 플레이 |
| **디바이스** | PC 브라우저 우선, 모바일 대응 (가상 조이스틱) |

### 1.4 플랫폼 및 출시 목표

- **플랫폼**: 브라우저 기반 (React SPA) — 별도 설치 불필요
- **서버**: Firebase Firestore (실시간 멀티플레이, 랭킹, 서버 통계)
- **저장 방식**: localStorage (개인 진행 상황) + Firestore (공개 랭킹, 멀티플레이)
- **목표 동시접속**: 채널 10개 × 최대 20명 = 200명 동시접속

### 1.5 개발 기간 및 팀 구성

| 역할 | 담당 |
|------|------|
| 기획 / 디자인 | 1명 |

## 2. 게임 컨셉 (Game Concept)

### 2.1 게임 소개 (엘리베이터 피치)

Tidehaven은 브라우저에서 즐기는 **멀티플레이어 힐링 낚시 RPG**입니다. 플레이어는 작은 해안 마을 Tidehaven에 도착해 낚시, 채굴, 요리, 허브 채집 등의 활동을 통해 캐릭터를 성장시키고, 희귀 물고기를 도감에 채우며, 서버 전체 플레이어와 랭킹을 경쟁합니다.

복잡한 전투 없이도 **능력치 그레이드업, 칭호 획득, 아이템 강화**라는 뚜렷한 성장 목표가 있어 "방치하다 돌아와도 즐거운" 게임 리듬을 추구합니다.

### 2.2 핵심 재미 요소 (Core Fun)

| # | 요소 | 설명 |
|---|------|------|
| 1 | **수집의 즐거움** | 22종 물고기 도감 완성, 희귀어 낚시의 짜릿함 |
| 2 | **성장의 보람** | 6가지 어빌리티 0→100 달성 + 그레이드업 반복 성장 |
| 3 | **경쟁과 과시** | 최대어 리더보드, 칭호 랭킹 Top 30 |
| 4 | **소셜 연결** | 채널별 최대 20인 동시 접속, 전설어 서버 공지 |
| 5 | **힐링 방치** | 오프라인 최대 2시간 보상, 자동 낚시 루프 |
| 6 | **경제 전략** | 요리 → 가격 3.5배 증폭, 낚싯대 강화 투자 |

### 2.3 레퍼런스 게임 (벤치마킹)

| 게임 | 참조 요소 |
|------|----------|
| **Stardew Valley** | 자원 채집 루프, 마을 NPC 관계, 힐링 분위기 |
| **MapleStory** | 어빌리티 그라인드, 칭호 시스템, 채널 멀티플레이 |
| **Fishing Planet** | 어종별 크기 편차, 진행 메커닉 |
| **Runescape** | 스킬 레벨업(0~99), 방치형 그라인드 허용 |
| **Animal Crossing** | 도감 완성 동기부여, 일일 보상 루틴 |

### 2.4 차별점 (Why This Game)

1. **설치 없이 즉시 플레이** — 브라우저 접속 즉시 시작, 닉네임만 입력
2. **채널 기반 소셜** — 10채널 × 20인, 같은 채널에서 타인 캐릭터 실시간 관찰
3. **날씨 연동 낚시** — 비 오는 날 낚시 확률 1.35배, 폭풍 시 낚시 불가 (전략적 선택)
4. **그레이드업 시스템** — 100 달성 후 초기화하는 무한 성장 구조 (희귀어 확률 보너스 최대 +80%)
5. **서버 전체 방송** — 전설/신화급 물고기 포획 시 전 채널 실시간 공지

### 2.5 게임 분위기 / 톤 앤 매너

- **비주얼**: 픽셀 아트 기반 탑다운 뷰, 따뜻한 파스텔 계열 팔레트
- **분위기**: 조용한 어촌 마을, 파도 소리, 나무 데크의 낚시터
- **사운드**: Web Audio API로 생성한 절차적 효과음 — 낚시 시작 물튀김 노이즈, 등급별 피치 변화 포획음, 레벨업 상승 멜로디 등 8가지
- **UX 기조**: "느긋하게, 꾸준히" — 강제 이벤트 없이 플레이어가 페이스 결정

## 3. 게임플레이 (Gameplay)

### 3.1 기본 조작 방식

| 입력 | PC | 모바일 |
|------|-----|--------|
| 이동 | 방향키 (↑↓←→) | 가상 조이스틱 (좌하단) |
| 낚시 시작 | `!낚시` 명령어 | 하단 [낚시] 버튼 |
| 채굴 시작 | `!광질` 명령어 | 하단 [광질] 버튼 |
| 채집 시작 | `!채집` 명령어 | 하단 [채집] 버튼 |
| 활동 중지 | `!그만` 또는 방향키 입력 | 방향키 또는 중지 버튼 |
| 요리 | `!요리` 명령어 (요리소 근처) | 하단 [요리] 버튼 |
| 판매 | `!판매` 명령어 (상점 근처) | 하단 [판매] 버튼 |
| 인벤토리 | `!인벤` | 하단 메뉴 버튼 |
| 상점 | `!상점` 명령어 또는 NPC 상호작용 | 상점 문 근처 접근 |
| 채팅 입력 | 하단 텍스트 입력창, Enter 전송 | 동일 |

**이동 물리 상세**: 최대 속도 3.5px/프레임, 마찰계수 0.80, 가속도 0.5. 질풍 신발 장착 시 +3.2px 추가.

### 3.2 핵심 루프 (Core Loop)

```
[마을 이동]
    │
    ├─► [낚시터 이동] → [낚시 의자 착석] → [!낚시] → [8~15초 대기] → [물고기 획득]
    │                                                      ↓
    │                                               [어빌리티 경험치 획득]
    │
    ├─► [광산 이동] → [광산 구역 진입] → [!광질] → [5~35초 대기] → [광석 획득]
    │                                                   ↓
    │                                           [강화 재료 누적]
    │
    ├─► [숲 이동] → [삼림 구역 진입] → [!채집] → [3~15초 대기] → [허브 획득]
    │
    ├─► [상점/요리소 이동]
    │       ├─► 상점: 장비 구매 / 판매
    │       └─► 요리소: !요리 → 물고기 가격 2~3.5배 증폭
    │
    └─► [어빌리티 100 달성] → [그레이드업] → [희귀어 확률 상승 영구 보너스]
                                   └─► 반복 성장 루프
```

**반복의 핵심**: 낚시 → 판매 → 골드 → 장비 강화 → 더 귀한 물고기 → 랭킹 상승

### 3.3 세션 길이

| 플레이 스타일 | 권장 세션 | 주요 활동 |
|-------------|---------|---------|
| 초단기 (5분) | 오프라인 보상 수령 + 판매 | 접속 보너스 300G + 오프라인 최대 960G |
| 단기 (15~30분) | 낚시 그라인드 + 퀘스트 완료 | 일일 퀘스트 3종 완수 |
| 중기 (1~2시간) | 광질 → 낚싯대 강화 → 고급 낚시 | 어빌리티 성장 + 칭호 변경 |
| 장기 (방치) | 오프라인 보상 누적 | 최대 2시간 × 8G/분 = 960G |

### 3.4 난이도 곡선

```
낚시 어빌리티  0~ 9 : 붕어·잉어·미꾸라지·메기 (흔함, 15~40G)
낚시 어빌리티 10~19 : 멸치·배스·송어 추가 (보통, 18~90G)
낚시 어빌리티 20~29 : 꽁치·강꼬치고기·황다랑어·금눈돔 (보통~희귀, 110~280G)
낚시 어빌리티 30~39 : 연어·황금붕어·오징어·낙지 (희귀, 260~380G)
낚시 어빌리티 40~49 : 참치·광어·감성돔 (희귀~전설, 450~650G)
낚시 어빌리티 50~69 : 우럭·뱀장어·황새치 (전설, 750~1000G)
낚시 어빌리티 70+   : 용고기·고대어 해금 (신화, 1800~3000G)
```

낚싯대 강화(0~100레벨)로 추가적인 수직 성장 가능. 강화 100레벨 시 포획 시간 40% 단축, 판매가 30% 상승.

## 4. 세계관 & 스토리 (World & Story)

### 4.1 배경 세계관

Tidehaven은 바다와 육지가 만나는 조용한 어촌 마을입니다. 북쪽에는 고대 광맥이 숨겨진 암석 산지가 펼쳐지고, 서쪽에는 희귀 약초가 자라는 울창한 숲, 남쪽에는 잔잔한 만과 깊은 바다로 이어지는 긴 부두가 있습니다.

마을 사람들은 예로부터 바다의 은총에 기대어 살아왔습니다. 그 중에서도 깊은 바다 끝에 산다는 **고대어(3000G)**와 **용고기(1800G)** — 이 두 신화적 존재는 Tidehaven의 전설이자 모든 낚시꾼의 꿈입니다.

> *"조류가 바뀌는 새벽, 안개 속 부두 끝에서 낚싯줄이 팽팽해지면 — 그것이 Tidehaven이 당신을 부르는 방식입니다."*

### 4.2 주요 등장인물 / NPC

| NPC | 위치 | 외형 | 역할 | 대사 |
|-----|------|------|------|------|
| **민준** | 상점 (Shop) | 갈색 피부, 검은 머리 | 장비 상인 | "어서오세요! 낚시장비라면 저한테 맡기세요~" |
| **수연** | 주방 (Cooking) | 붉은 옷, 검은 머리 | 요리 조수 | "오늘은 어떤 요리를 만들어볼까요? 😊" |
| **미나** | 여관 (Inn) | 분홍 옷, 갈색 머리 | 여관 주인 | "어서오세요! 쉬고 싶으시면 말씀해주세요~ 🌙" |
| **철수** | 광산 내부 (Mine) | 회청색 옷, 검은 머리 | 광부 안내인 | "이 광산엔 귀한 광석이 많답니다! 조심하세요" |

### 4.3 메인 스토리 라인

현재 버전에서는 명시적 메인 스토리보다 **플레이어 자신의 성장 여정**이 서사의 중심입니다.

- **챕터 0 — 입항**: 닉네임을 가진 여행자로 Tidehaven에 도착. 초급 낚시대 하나와 100G로 시작.
- **챕터 1 — 정착**: 낚시와 채굴로 생계를 마련하고 마을 NPC와 친해지기.
- **챕터 2 — 성장**: 낚시 어빌리티 40 돌파, 중급 낚시대 구매 (500G + 재료), 전설급 어종 첫 포획.
- **챕터 3 — 전설**: 낚시 어빌리티 70 달성, 신화 어종 해금, "낚시의 신" 칭호 획득 도전.
- **최종 목표**: 22종 물고기 도감 완성 → "박물학자" 칭호 획득 + 서버 전체 방송.

🔜 **예정**: NPC 개별 퀘스트 스토리라인, 계절별 이벤트 스토리, 전설어 신화적 세계관 확장.

### 4.4 지역별 설정

| 구역 | 타일 좌표 | 분위기 | 주요 활동 |
|------|---------|-------|---------|
| **마을 광장** | col 0~29, row 12~17 | 따뜻한 초록 잔디, 황토 포장길 | 이동, NPC 대화, 건물 입장 |
| **해변 & 모래사장** | row 18~20, col 0~44 | 황금빛 모래, 파도 경계 | 낚시터 진입 통로 |
| **메인 낚시터** | row 21~22, col 3~42 | 나무 데크 (8개 의자) | 일반 낚시 |
| **심해 방파제** | row 23~26, col 20~30 | 깊은 바다 뷰 (4개 의자) | 심해 낚시 (가격 ×1.5) |
| **숲 구역** | col 30~44, row 0~18 | 짙은 녹색 캐노피, 바위 산재 | 허브 채집 |
| **광산 구역** | col 45~69, row 0~22 | 회색 암반 | 채굴, 광산 건물 입장 |
| **수역** | row 27+, col 0~44 | 애니메이션 파도, 반짝임 | 배경 (진입 불가) |

### 4.5 게임 내 시간 / 날씨 설정

날씨는 **채널별로 결정론적(deterministic)**으로 생성됩니다. 같은 채널의 모든 플레이어가 동일한 날씨를 경험하며, **8분마다** 자동 변경됩니다.

| 날씨 | 아이콘 | 발생 확률 | 낚시 배율 | 채굴 배율 | 어빌리티 배율 | 특이사항 |
|------|------|---------|---------|---------|-----------|--------|
| 맑음 | ☀️ | 40% | ×1.00 | ×1.00 | ×1.0 | 기본 상태 |
| 흐림 | ⛅ | 25% | ×1.10 | ×1.00 | ×1.0 | 약간 유리 |
| 비 | 🌧️ | 20% | ×1.35 | ×0.90 | ×1.2 | 낚시 황금 시간 |
| 폭풍 | ⛈️ | 5% | ×0.00 | ×0.80 | ×0.5 | **낚시 완전 불가** |
| 안개 | 🌫️ | 10% | ×0.80 | ×0.75 | ×0.9 | 전반 디버프 |

## 5. 캐릭터 시스템 (Character System)

### 5.1 캐릭터 생성

로그인 화면에서 아래 항목을 설정한 뒤 입장합니다.

| 항목 | 선택지 | 기본값 |
|------|-------|-------|
| **닉네임** | 2~12글자 | 필수 입력 |
| **성별** | 남성 / 여성 | 남성 |

### 5.2 성장 시스템

#### 어빌리티 성장 공식

6개 어빌리티 모두 `0.00 ~ 100.00` 범위에서 소수점 2자리로 관리됩니다.

| 어빌리티 | 아이콘 | 성장 방법 | 1회 획득량 | 주요 효과 |
|---------|------|---------|---------|---------|
| **낚시** | 🎣 | 물고기 낚기 | 흔함 +0.30 / 보통 +0.60 / 희귀 +1.10 / 전설 +2.00 / 신화 +3.50 | 포획 속도 단축, 희귀어 확률 상승, 판매가 상승 |
| **채굴** | ⛏ | 광석 채굴 | 철광석 +0.40 / 구리광석 +0.75 / 수정 +1.30 | 채굴 속도 단축 |
| **요리** | 🍳 | 요리 완료 | 물고기 1마리당 +0.20 | 요리 배율 추가 상승 (최대 +1.0배) |
| **화술** | 💬 | 물고기 판매 | 100G당 +0.08 | 판매가 최대 +50% 상승 |
| **체력** | ❤️ | 모든 활동 완료 | 활동마다 +0.25 (허브 채집 +0.10) | 모든 활동 속도 단축 |
| **강화** | 🔨 | 낚싯대 강화 성공 | 성공 1회당 +0.80 | 강화 성공률 최대 +30% 상승 |

#### 그레이드업 (Grade-Up)

어빌리티가 **100.00**에 도달하면 그레이드업 가능 상태가 됩니다.

- 스킬창에서 그레이드업 확인 → 해당 어빌리티 수치를 0.00으로 초기화
- **그레이드 숫자**는 영구 누적 (Grade 0 → 1 → 2 → ...)
- **희귀어 가중치 영구 보너스**: Grade × 10% (최대 +80%)
  - Grade 1: +10% / Grade 2: +20% / Grade 3: +30% / ... / Grade 8+: 상한 +80%
  - 효과 분배: 보통 +50%, 희귀 +100%, 전설 +150%, 신화 +200% (각 그레이드 보너스 내)

#### 파티 보너스 ✅ 구현됨

같은 채널에 다른 플레이어가 1명 이상 존재할 경우, 어빌리티 획득 시 **20% 확률로 획득량 2배** 적용.

## 6. 콘텐츠 시스템 (Content Systems)

### 6.1 낚시 시스템

#### 낚시 시작 조건

1. 낚시 의자 3타일 반경 이내 위치
2. 해당 의자를 다른 플레이어가 점유하지 않을 것
3. 날씨가 폭풍(⛈️)이 아닐 것
4. `!낚시` 명령어 입력

#### 낚시 시간 계산

```
기본 시간 범위: 낚싯대별 catchTimeRange
시간 배율 = max(0.3, (1 - 낚시 × 0.004) × (1 - 체력 × 0.003) × (1 - 강화효과))
최종 시간 = randInt(기본min × 배율, 기본max × 배율)  최솟값 1초
```

| 낚싯대 | 기본 시간 범위 | 시간 배율 | 구매 조건 |
|-------|------------|--------|---------|
| 초급 낚시대 | 8.0~15.0초 | ×1.00 | 기본 제공 |
| 중급 낚시대 | 6.0~12.0초 | ×0.85 | 500G + 철광석×3 + 구리광석×2 + 초급 강화 50 달성 |
| 고급 낚시대 | 4.0~9.0초 | ×0.70 | 2000G + 철광석×5 + 수정×3 + 중급 강화 50 달성 |

#### 판매가 계산

```
기본가    = FISH[name].price
크기배율  = 실제크기 / 평균크기  (minSz~maxSz 범위 랜덤)
강화보너스 = 강화레벨 × 0.003
최종가    = round(기본가 × 크기배율 × (1 + 화술 × 0.005 + 낚시 × 0.002 + 강화보너스))
심해보너스 = 심해 방파제 의자 낚시 시 × 1.5배
```

| 이벤트 | 발생 확률 | 내용 |

### 6.2 광질 시스템

#### 채굴 조건

- 광산 구역 (col 45~69, row 0~22) 내 위치
- `!광질` 명령어 입력

#### 광석 테이블

| 광석 | 색상 | 출현 가중치 | 채굴 시간 | 판매가 | 주요 용도 |
|------|------|-----------|---------|------|---------|
| 철광석 | 회색 | 50 | 5~10초 | 30G | 중급낚시대, 고급화로, 강화 재료 |
| 구리광석 | 구리색 | 30 | 7~14초 | 50G | 중급낚시대, 강화 재료 |
| 수정 | 하늘색 | 20 | 12~22초 | 100G | 고급낚시대, 질풍신발, 강화 재료 |
| 금광석 | 금색 | 8 | 18~35초 | 350G | 판매 전용 (고수익) |

광석은 상점에서 장비 제작/구매 재료로 사용됩니다.

| 아이템 | 필요 골드 | 필요 광석 | 효과 |
|-------|---------|---------|------|
| 중급 낚시대 | 500G | 철광석×3, 구리광석×2 | 포획시간 배율 ×0.85 |
| 고급 낚시대 | 2000G | 철광석×5, 수정×3 | 포획시간 배율 ×0.70 |
| 질풍신발 | 1800G | 수정×3 | 이동속도 +3.2px/프레임 |
| 고급화로 | 900G | 철광석×3 | 요리 배율 ×3.5 |

#### 낚싯대 강화 시스템

낚싯대는 0에서 100레벨까지 강화할 수 있습니다. 강화는 상점/스탯 패널의 장비 탭에서 실행.

| 항목 | 공식 / 수치 |
|------|-----------|
| 강화 비용 | `50 + 강화레벨 × 20G` (레벨 0 = 50G, 레벨 99 = 2030G) |
| 성공률 | `max(0.20, 1.0 - 레벨 × 0.008) + 강화어빌리티 × 0.003` |
| 성공률 예시 | 레벨 0: 100% / 레벨 50: ~60% / 레벨 100: 최소 20% |
| 강화 재료 (레벨 0~29) | 없음 |
| 강화 재료 (레벨 30~59) | 철광석 1개 |
| 강화 재료 (레벨 60~79) | 구리광석 1개 |
| 강화 재료 (레벨 80~100) | 수정 1개 |
| 포획 시간 감소 효과 | 레벨 × 0.4% (레벨 100 = 40% 단축) |
| 판매가 상승 효과 | 레벨 × 0.3% (레벨 100 = 30% 상승) |

중급/고급 낚싯대 구매를 위해서는 이전 낚싯대의 **강화 레벨 50 달성이 필수 조건**입니다.

### 6.4 요리 시스템

#### 요리 조건

- 요리소 건물 4타일 반경 이내 위치
- 요리도구 보유 (`기본프라이팬` 또는 `고급화로`)
- `!요리` 명령어 입력

#### 요리 배율 계산

```
총 요리 배율 = 기본배율 + 요리어빌리티 × 0.01

기본프라이팬: 2.0 + (요리 × 0.01)  → 최대 3.0배 (요리 100 달성 시)
고급화로:    3.5 + (요리 × 0.01)  → 최대 4.5배 (요리 100 달성 시)
```

요리 후 인벤토리 내 물고기 전체에 `cooked: true` 플래그가 부여되며 판매가가 영구 반영됩니다.

### 7.5 인플레이션 방지 설계

1. **1회용 미끼 소비 구조**: 전설 미끼 600G, 신화 미끼 3,000G — 사용 즉시 소모되는 강력한 골드 싱크
2. **강화 비용 누적**: 레벨이 높아질수록 1회 비용 증가 (레벨 99 = 2,050G/회) + 실패 가능성
3. **요리도구 초기 비용**: 프라이팬 200G, 화로 900G + 재료 — 수익 배율 향유에 선투자 필요
4. **낚싯대 진행 잠금**: 중급→고급 구매에 이전 낚싯대 강화 레벨 50 필수 → 강화 누적 비용 강제
5. **오프라인 보상 상한**: 최대 2시간(960G) — 무한 방치 수익 방지
6. **허브 용도 미확정**: 현재 허브는 판매 전용 (추후 요리 레시피 연계 예정)

### 8.1 채팅

### 10.3 이동 방식 및 동선

#### 이동 물리 수치

| 항목 | 수치 |
|------|------|
| 기본 최대 속도 | 3.5px/프레임 |
| 가속도 | 0.5px²/프레임 |
| 마찰계수 | 0.80 (키를 놓으면 서서히 감속) |
| 빠른 신발 최대 속도 | 5.0px/프레임 (+1.5) |
| 질풍 신발 최대 속도 | 6.7px/프레임 (+3.2) |
| 캐릭터 히트박스 | 18×26px (4모서리 충돌 검사) |

#### 권장 동선

```
[시작: col15, row14]
│
├─► 남쪽 이동 (row↑) → 해변(row18) → 메인 낚시터(row22) → 낚시 의자 착석
│
├─► 동북 이동 → 숲(col30~44) → 허브 채집
│
├─► 동쪽 이동 → 광산(col45~) → 채굴
│
└─► 북쪽 이동 → 상점 문(col4.5, row11) → Enter → 실내 상점 진입
                                                    → 장비 구매 / 판매
```
